# nutrition_app
This application is designed for users, farmers, and nutritionists, supporting three roles: USER, FARMER, NUTRITIONIST.

Built with:

Backend: Node.js + NestJS (TypeScript)
Frontend: React + Tailwind CSS
Database: MySQL
Auth: JWT
👥 Roles
🧍 USER
register / login
manage preferences (allergies, dislikes, conditions)
manage fridge (ingredients)
receive recipes and meal plans
chat with nutritionist
view own data only
🚜 FARMER
register / login
add and manage products (price, stock)
marketplace access
🥗 NUTRITIONIST
register / login
view assigned users
access user data (preferences + fridge)
create meal plans and recommendations
chat with users
⚙️ Features
JWT authentication & role-based access
Fridge management
Nutrition profiles (goals, calories)
Recipes integration (Spoonacular API)
Meal plans
Chat system
Marketplace
